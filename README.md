# 📝 Quiz App

A simple quiz application built with JavaScript that allows users to answer questions and track their progress.

## 📚 About The Project
This project is part of my learning journey in JavaScript.  
It focuses on building interactive quizzes, handling user input, and managing application state.

## 🚀 Live Demo
👉 [View Live Demo](https://ali-haitham00.github.io/quiz-app-js/)

## 🚀 Features
- Multiple choice questions
- Questions counter and category display
- Timer for each question
- Progress indicator
- Submit answers and navigate through questions
- Clean and user-friendly UI
